Chapter 4: Core Actions in Neurovectors

Introduction

Core actions like training and testing are executed within a project. This chapter explains how to create a project, load a dataset, train a model, and run tests.

Create a Project and Load a Dataset

Create a project and load a dataset:

from neurovectors import Proyecto

# Create a new project
project = Proyecto(titulo="Example Project", verbose=True)

# Load a dataset into the project
dataset = project.add_dataset(name="demo_dataset", file="data.csv")

# Define study variables
dataset.set_study(["column1", "column2"])

The dataset is initialized, rows/columns are loaded, and study variables selected.

Training

The train method trains a model using a dataset and engine:

# Add an engine
engine = project.add_engine(name="engine_demo", type="EngineDict")

# Add analytics
analytics = project.add_analytics(name="analytics_demo", type="Clas")

# Train the model
project.actions.train(
    dataset=dataset,
    engine=engine,
    analytics=analytics,
    epochs=5,
    mode="success"
)

This uses 'success' mode, training only rows misclassified in previous epochs.

Testing

The test method evaluates the model with a dataset:

# Create a subset (optional)
sample_train, sample_test = project.add_split(
    name_part1="train", 
    name_part2="test", 
    rows=1000, 
    percentage=80
)

# Test the model
project.actions.test(
    dataset=dataset,
    engine=engine,
    analytics=analytics,
    filter_rows=sample_test
)

Here, test rows are defined using a previously created sample.

End-to-End: Train and Test

A full training/testing flow:

from neurovectors import Proyecto

# Create a project
project = Proyecto(titulo="End-to-End", verbose=True)

# Load a dataset
dataset = project.add_dataset(name="full_dataset", file="data.csv")
dataset.set_study(["column1", "column2"])

# Add engine and analytics
engine = project.add_engine(name="full_engine", type="EngineDict")
analytics = project.add_analytics(name="full_analytics", type="Clas")

# Create train/test samples
sample_train, sample_test = project.add_split(
    name_part1="train", 
    name_part2="test", 
    rows=1000, 
    percentage=80
)

# Train the model
project.actions.train(
    dataset=dataset,
    engine=engine,
    analytics=analytics,
    filter_rows=sample_train,
    epochs=10,
    mode="all"
)

# Test the model
project.actions.test(
    dataset=dataset,
    engine=engine,
    analytics=analytics,
    filter_rows=sample_test
)

# Show project information
project.show_project()

Additional Details

  • train may auto-stop when 100% accuracy is achieved.
  • Training modes ('success', 'all', 'group') adapt to different needs.
  • Testing can run on the full dataset or specific samples.
