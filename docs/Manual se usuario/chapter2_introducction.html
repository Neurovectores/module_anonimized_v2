Chapter 2: Project Structure and Basic Usage

Introduction

This chapter shows how to create and structure a project in Neurovectors. Projects have four core components: datasets, engines, samples, and analytics. Below are the initial steps to work with a project.

Create a New Project

Use the Proyecto class as follows:

from neurovectors import Proyecto

# Create a project
project = Proyecto(titulo="My Example Project", verbose=True)

# Show general project info
project.show_project()

Basic Project Structure

A Neurovectors project consists of:

  • Datasets: Data files used for training and analytics.
  • Engines: Components that perform predictions/calculations.
  • Samples: Randomly selected subsets for training or testing.
  • Analytics: Analysis and evaluation methods.

Manage Datasets

Add and manage datasets:

# Add a new dataset
dataset = project.add_dataset(name="sales_dataset", file="sales.csv")

# Show dataset details
dataset.show_dataset()

# Drop a dataset
project.drop_dataset(name="sales_dataset")

Manage Engines

Engines process data and perform predictions:

# Add a new engine
engine = project.add_engine(name="pred_engine", type="EngineDict")

# Show engine info
engine.show_info()

# Drop an engine
project.drop_engine(name="pred_engine")

Manage Samples

Samples split data into random subsets:

# Create two samples
sample1, sample2 = project.add_split(
    name_part1="train", 
    name_part2="test", 
    rows=1000, 
    percentage=80
)

# Show a sample
print(project.get_split(name="train"))

# Drop a sample
project.drop_split(name="train")

Manage Analytics

Analytics provide metrics and evaluations:

# Add an analytics
analysis = project.add_analytics(name="hits_analysis", type="Clas", method="Hits")

# Show analytics info
analysis.show_info()

# Drop analytics
project.drop_analytics(name="hits_analysis")

Full Example

A complete example to create and structure a project:

from neurovectors import Proyecto

# Create a project
project = Proyecto(titulo="Full Project", verbose=True)

# Add a dataset
dataset = project.add_dataset(name="demo_dataset", file="data.csv")

# Add an engine
engine = project.add_engine(name="demo_engine", type="EngineDict")

# Create samples
sample_train, sample_test = project.add_split(
    name_part1="train", 
    name_part2="test", 
    rows=1000, 
    percentage=80
)

# Add analytics
analysis = project.add_analytics(name="demo_analysis", type="Clas", method="Hits")

# Show general project info
project.show_project()
