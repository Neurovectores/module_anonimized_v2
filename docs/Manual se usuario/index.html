Chapter 1: Introduction and Project Setup

Overview

Neurovectors is a toolkit for managing predictive projects. It provides advanced functionality to manipulate datasets, prediction engines, and analytics, as well as tools to inspect overall project information and estimate memory usage.

Main Features

  • Dataset management: add, drop, explore, and analyze data.
  • Creation and configuration of prediction engines.
  • Custom analytics per study variable.
  • Data splitting into random samples.
  • Project memory footprint monitoring.

Prerequisites

  • Python 3.8 or higher.
  • Dependencies:
    • pandas
scikit-learn
tqdm
babel
numpy
psutil
pandasgui
matplotlib
networkx
pyvis
  • Install dependencies:
    • pip install pandas scikit-learn tqdm babel numpy psutil pandasgui matplotlib networkx pyvis
  • Optional: a project file with .pkl extension to load an existing project.

Create a New Project

Use the Proyecto class to start a new project:

from neurovectors import Proyecto

# Create a new project
project = Proyecto(titulo='My Test Project', verbose=True)

# Show project information
project.show_project()

Internal Structure

The project is managed through two main classes:

  • O_Proyecto: Defines base configuration such as title, version, and seed.
  • Proyecto: Orchestrates project lifecycle, including loading/saving and interaction with datasets and engines.
