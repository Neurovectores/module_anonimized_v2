Chapter 5: Advanced Engine Usage

Introduction

The Engine is a core component using dicts and sets to manage nodes, neurovectors, and their connections. This chapter covers engine usage, its interactions with Actions and Dataset, and how training/predictions work.

Basic Engine Interaction

The engine is managed from a Proyecto and used via Actions. Typical flow:

  1. Create a project and dataset.
  2. Add an engine to the project.
  3. Convert dataset rows to neurovectors.
  4. Train the engine.
  5. Predict and test.
from neurovectors import Proyecto

# Create a project
project = Proyecto(titulo="Engine Usage", verbose=True)

# Load a dataset
dataset = project.add_dataset(name="engine_dataset", file="data.csv")
dataset.set_study(["column1", "column2"])

# Add an engine
engine = project.add_engine(name="main_engine", type="EngineDict")

Convert Rows to Neurovectors

Conversion is done via engine's conver_row method:

# Convert rows to neurovectors
total_rows = project.actions.convert(dataset=dataset, engine=engine)

This creates nodes and neurovectors from dataset data, connecting columns to values.

Engine Training

Training uses Actions.train which interacts with the engine:

# Train the engine
project.actions.train(
    dataset=dataset,
    engine=engine,
    analytics=project.add_analytics(name="engine_analysis", type="Clas"),
    epochs=5,
    mode="success"
)

During training, neurovectors, scope, and results are analyzed to adjust the model.

Predictions

The engine predicts by finding the most similar neurovectors to the stimulus:

# Make a prediction
result = engine.predict(
    analitycs=project.actions.analytics,
    stimulus={"column1": "value1", "column2": "value2"},
    scope=1
)

The stimulus can be a dict, pandas Series, or formatted string.

Helper Methods

The engine includes methods to inspect and manage neurovectors:

  • Show info:
    • # Show engine info
engine.show_info()
  • Show all data:
    • # Show all data in pandasgui
engine.show_all()
  • Graph a neurovector:
    • # Draw a specific neurovector
engine.paint_nv(nv=0)
  • Save/Load engine state:
    • # Save engine state
engine.save("engine_state.pkl")

# Load engine state
engine.load("engine_state.pkl")

End-to-End Flow

Combining conversion, training, and predictions:

# Create a project and engine
project = Proyecto(titulo="End-to-End", verbose=True)
dataset = project.add_dataset(name="full_dataset", file="data.csv")
dataset.set_study(["column1", "column2"])
engine = project.add_engine(name="full_engine", type="EngineDict")

# Convert rows to neurovectors
project.actions.convert(dataset=dataset, engine=engine)

# Train engine
project.actions.train(
    dataset=dataset,
    engine=engine,
    analytics=project.add_analytics(name="full_analysis", type="Clas"),
    epochs=10,
    mode="all"
)

# Run predictions/tests
project.actions.test(
    dataset=dataset,
    engine=engine,
    analytics=project.add_analytics(name="full_analysis", type="Clas")
)

# Inspect engine
engine.show_info()
engine.paint_nv(nv=0)
